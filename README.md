Sahyadri-Samrakshane is a mobile-based environmental reporting application designed to help citizens report ecological threats in sensitive forest regions such as the Western Ghats.
The application allows users to report incidents including forest fi res, landslides, illegal tree cutting, and wildlife sightings by capturing photos and location details.
The application acts as a bridge between local communities and forest authorities, enabling quick and accurate communication for environmental protection.

The vision of Sahyadri-Samrakshane is to create a real-time, location-aware ecological alert system for faster disaster response in forest regions (citizen-driven ecological alert system) that empowers people to actively participate in environmental protection.
The application aims to provide a fast, simple, and reliable reporting system for ecological incidents using smartphone capabilities such as camera and GPS.
